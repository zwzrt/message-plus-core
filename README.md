### 介绍

基于netty的消息增强器核心，帮助开发者快速开发 即时聊天系统。

功能：

	1、支持单发、群发、聊天室及系统功能。
	2、以及数据持久化。
	3、支持失败消息的持久化及重发功能。

### 软件架构

Maven + SpringBoot + Hutool + Netty

### 快速上手

   ```xml
   <dependency>
       <groupId>io.github.zwzrt</groupId>
       <artifactId>message-plus</artifactId>
       <version>0.2.2-beta</version>
   </dependency>
   ```
前往 [学习文档](https://www.red-coral.cn/nav)
